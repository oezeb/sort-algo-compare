1. 科学技术原理

Insertion Sort 插入排序

每次遍历时从无序数列中选择一个元素插入到有序数列中。重复该操作直到无序数列为空。

Selection Sort 选择排序

每次遍历当前范围元素选择最小或最大值与当前范围的第一个元素交换位置。此后再在第一个元素之后的范围重复该操作。

Bubble Sort 冒泡排序

每次遍历的过程中对相邻元素两两比较，若该元素对为逆序则交换位置。重复该操作直到数列中不存在逆序对

Merge Sort 归并排序

先将序列（递归地）分为两半分别排序最后将结果归并起来。

Quick Sort 快速排序

选择一个基准元素,将待排序列分成两部分,一部分比基准元素小,一部分大于等于基准元素,此时基准元素在其排好序后的正确位置,然后再用同样的方法递归地排序划分的两部分。

Heap Sort 堆排序

跟选择排序原理相同。因为当前范围为二叉堆选择最小或最大元素更快（二叉堆根节点的值最小或最大)。

3. 创新性描述

  • 原创性： 纯原创
  • 优点与特色：
    • 实时通过Spinner调动画运行速(运行时也能调动画速)。
    • 在AlgoDetailsView、动画执行时禁止更改Graph的数据。于是利用了多线程等待动画执行结束再允许改数据。

4. 运行方法和参数设置

  • 运行方法

    • Home View
      • 点击任何Graph执行对应的数据和算法的动画。
      • 通过PLayAll按钮将全部Graph同时执行。
      • 通过各列（行）按钮同时执行对应的列（行）Graph。
      • 点击Reset按钮将全部Graph数据移回原位置。
      • 在代表各排序算法的按钮右上角、点击小按钮显示对应算法的AlgoDetailsView界面。
    • AlgoDetailsView
      • 点击PLay按钮执行。
      • 点击Reset按钮将Graph数据移回原位置。

  • 参数设置

    • 通过Spinner右方的上下按钮或键盘的上下按钮调Spinner数值字段（调动画执行速度）。
    • 在AlgoDetailsView通过Graph上面的数值字段改Graph中每个元素的值。
